# ResoniteBaballoniaEyeTracking

> [!NOTE]
> Resonite officially supports eye tracking by Baballonia from 2026.3.3.1232, so this mod is no longer necessary.

A [ResoniteModLoader](https://github.com/resonite-modding-group/ResoniteModLoader) mod for [Resonite](https://resonite.com/) that enable [Baballonia](https://github.com/Project-Babble/Baballonia) EyeTracking.
